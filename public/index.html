Whats in this website?
  • Can mutiple quiz rooms at any given time
  • Whole students records based on quiz such as correct, wrong and skipped answers.
How this website is build
  • Frontend: HTML, CSS, JS
  • Whole UI chnage is with help of JS methods, not with (.innerHTML)
  • Backend: NodeJs
  • No database added. Once server is closed, whole data will be wiped off
  • For real time quizzes and rooms, Socket.io is used
  • Backend API's:
    • GET: BASE_URL/student/performance
    • POST: BASE_URL/student/join-quiz-room
    • GET: BASE_URL/teacher/get-question
    • GET: BASE_URL/teacher/performance
    • POST: BASE_URL/teacher/question-answered
    • POST: BASE_URL/teacher/question-skipped
    • POST: BASE_URL/teacher/create-quiz-room

    BASE_URL: https://quiz-cp.herokuapp.com
